## 8. Motion & interaction  `☑`
*Why it matters: animation level dramatically changes the feel (and the prompt).*
- **Overall motion level:** expressive but tasteful (Apple-quality, smooth).
- **Signature effect:** vertical scroll *reads as lateral progression* — as you scroll down, sections **slide/drift sideways** into view (and reverse on scroll-up), so it feels like moving side-to-side through panels. Theme/tint shifts subtly between panels.
- **Aggressiveness:** **SUBTLE — sections drift sideways but normal scrolling still works** (no full scroll-hijack). `locked` *(Apple-like reveal, not a scroll trap — protects mobile/recruiter usability.)*
- **Other:** liquid-glass hover states, smooth eased transitions, soft parallax on glass layers.
- **Guardrails:** keep it GPU-light enough to stay smooth on a mid-range phone; respect `prefers-reduced-motion`; don't break the back button or normal scroll.

## 9. Inspiration & anti-patterns  `☑`
*Why it matters: 2–3 reference sites teach Claude Design more than paragraphs of adjectives.*
*(No single live site has all four traits — green + glass + lateral-scroll + portfolio — so mix-and-match: borrow one thing from each.)*

**Reference sites — study these, borrow the specific thing noted:**
- **Apple — Liquid Glass / product pages** (apple.com newsroom "Liquid Glass" + any iPhone product page) → the glass *material* itself: frosted translucency, specular edge highlights, soft inner glow, floating rounded panels, nav that materializes on scroll. **This is the anchor for the look.**
- **Heights** (heights.agency) → green-as-background with white text; green gradient depth + how white type stays legible on green. *(Their green is teal/saturated — substitute our sage.)*
- **Jesse Martinez** (jessemartinezdesign.com) → dark base + green accent on a real single-dev portfolio; scroll-triggered section reveals.
- **Eduard Bodak** (eduardbodak.com) → the scroll *feel*: smooth GSAP/Lenis-style motion + scroll-progress, tasteful (no hijack). Borrow pacing, not colors.
- **Fleur Moreau** (fleurmoreau.com) → vertical scroll → **lateral/horizontal section transitions** = our signature effect. Borrow the sideways motion.
- **Cole template** (derekcole.framer.website) → clean multi-section portfolio skeleton (hero → project list → about → connect).

**Hard "do NOT do this" list:**
- No full scroll-hijack — normal scrolling must keep working.
- Don't use the reference greens (teal/lime) — use our sage palette only.
- Don't put low-contrast white text on the *light* sage panels — meet WCAG 4.5:1 (use dark green text there).
- Don't over-blur / over-animate to the point it lags on a mid-range phone.
- Apple's true Liquid Glass can't be reproduced 1:1 in CSS — approximate with `backdrop-filter` blur + translucent tint + subtle borders; keep it tasteful.
